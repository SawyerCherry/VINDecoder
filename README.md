# VINDecoder
This app decodes DTC and VIN codes using two separate API's.
This app is built in SwiftUI, and Swift
How to Use:
After the simulator is built, take your VIN (Vehicle Identification Number) from your vehicle, it can be found usually on the Driver side window below the wiper, or on your title. Enter in the VIN and hit "Fetch Data", depending on the vehicle, your data should display. It usually depends whether or not the API has certain objects in it. But for testing, you should use the VIN that I provided in the comments on ContentView.swift, and do the same thing for the DTC code and 11 Digit VIN that I provided on the DiagnosticView.swift.
